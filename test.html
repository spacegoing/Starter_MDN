you are calculating the specificity the wrong way.

lets see:

you have 2 rules separated by a comma , the first one has 1 ID(box1), 1 class(.spec2) and 2 elements div and p, which makes(0,1,1,2)

the second one has 2 IDs and 1 element, which makes (0,2,0,1)

#box1 div.spec2 p,
    #box1 #box2 p {
        color: blue;
    }

This one is exactly like your second rule above (0,2,0,1), which means it won't do anything because the first rule(0,1,1,2) is more specific than this one. 

#box1 #box3 p {
    color: green;
}

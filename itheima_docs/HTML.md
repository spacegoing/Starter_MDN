# HTML Basics #

## Tags ##

### Anchor <a href=""></a> ###

#### 基本用法 ####

1. `href=""` 中 `value` 为 `url`: 跳转到 `url`
2. `href=""` 中 `value` 为 `#`: 跳转到当前页面
3. `href=""` 中 `value` 为 `#id`: 跳转到 `id` 元素
4. `href=""` 中 `value` 为 `url#id`: 在跳转到的页面进行定位

#### 其他属性 ####

- `target=""`
    - `_blank`: 打开新页面同时保存原有页面
    - `_self`: 默认值，不打开新页面

### Base <base /> ###

- `<base target="_blank" />`: 给所有 `<a>` 设置跳转方式
